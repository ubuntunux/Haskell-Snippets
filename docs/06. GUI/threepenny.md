examples

```
git clone https://github.com/HeinrichApfelmus/threepenny-gui.git
cd threepenny-gui/
stack init
stack setup
stack build
cd samples
./runhaskell Mouse.hs
```

After you have started an example, open your web browser and point it to the address [http://localhost:8023](http://localhost:8023). Enjoy!
