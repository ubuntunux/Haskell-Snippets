### 패턴매칭
간단한 팩토리얼 함수, 하지만 아래와 같이 작성했다가는 프로그램이 절대로 끝나지 않는다. 왜냐하면 x값이 1이되더라도 아랫쪽에 패턴매칭이 되기전 fac x 함수에 매칭이 되기 때문에 무한반복에 빠진다.
```
fac x = x * fac (x-1)
fac 1 = 1
```
패턴매칭은 위쪽부터 이루어지기 때문에 이점을 고려해서 작성해야 한다. 아래와 같이 작성하자
```
fac 1 = 1
fac x = x * fac (x-1)
```
